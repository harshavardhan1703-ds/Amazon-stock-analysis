# Amazon Stock Price Analysis & Prediction

## Overview
This project analyzes Amazon stock price history using Python and Machine Learning techniques.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn
- Jupyter Notebook

## Features
- Data Cleaning & Preprocessing
- Moving Average Analysis
- Bollinger Bands
- Volatility Analysis
- Trading Volume Analysis
- Linear Regression Stock Price Prediction

## Machine Learning Model
- Linear Regression

## Project Goal
To analyze historical Amazon stock data and predict future stock closing prices using machine learning techniques.
